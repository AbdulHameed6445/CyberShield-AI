# 🛡️ CyberShield AI

CyberShield AI is an intelligent phishing email detection and threat analytics platform built using Machine Learning, Natural Language Processing (NLP), and Streamlit.

## Features

* 📧 Phishing Email Detection
* 🧠 NLP-Based Content Analysis
* 🔍 Suspicious Keyword Detection
* 🌐 URL Extraction
* 🎯 Threat Score Calculation
* 🤖 AI Security Analyst
* 📄 Automated Incident Reports
* ⬇ Downloadable Reports
* 📊 Security Analytics Dashboard
* 📁 Email File Upload Support

## Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas
* Plotly
* NLP
* Machine Learning

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
